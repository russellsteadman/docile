What is Docile?

Docile is an open source JavaScript library for use in the browser.
Docile can:

  • Store and retrieve data about nodes, including objects. Learn More
  • Store and retrieve "links" to other nodes. Learn More
  • Retrieve the data of "linked" nodes. Learn More

How can I use Docile in my project?

Docile is simple to download.

  • npm install -S docile
  • yarn add docile
  • bower install -S docile
    • Download From GitHub Unpkg CDN
